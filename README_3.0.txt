名刺データ抽出ツール（ver3.0以降）

・tool_verX.X.jar
	ツール本体です。
	起動すればそのまま使えます。

	【操作手順】
	1．ツールを起動する。

	2．以下のどちらかからデータの抽出方法を選択する
		①前回取り込んだ日付以降のデータを抽出する（ver3.0では未実装）
		　ログファイルの情報から最後にデータを抽出した日付以降のデータを抽出します。

		②日付を指定する
		　指定した期間に取り込まれた名刺データを抽出することができます。
		　①と違い、最近取り込んだデータ以外のデータ以外も抽出することが可能です。
		　最新のデータ以外（過去に取り込んだデータ）を抽出するときには
		　右側の「最新データ」のチェックを外してください。（ログファイルへの記録内容が変わります）

	3．抽出したい名刺データの含まれたCSVファイルをドラッグアンドドロップする。


	以下注意点
	・一度にドロップできるファイルは1つだけです。
	・名の列は空白になる仕様です。
	・郵便番号のデータから住所を探索しているため、変換に時間がかかります。
	・郵便番号のデータにない郵便番号の場合、都道府県より後の部分はすべて市区郡の列に登録されます。
	・ログ内容が出力されるまで時間がかかるため、
	　読み込むデータが多いとフリーズしているように見えますが実際には動作しています。
	　エラーが出ているときにはエラーメッセージが出力されるはずですので、ログが出力されるまで待ってください。
